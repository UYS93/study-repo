JavaScript란 무엇인가?

JavaScript는 웹 페이지와 웹 기반 응용 프로그램을 만들기 위해 주로 사용되는 프로그래밍 언어입니다.
이는 클라이언트 사이드 스크립팅 언어로 서버 대신 사용자의 컴퓨터 또는 기기에서 코드가 실행됩니다.
JavaScript는 웹 사이트에 상호 작용성과 기타 동적 기능을 추가하는 데 사용할 수 있으며,
예를 들어 상호 작용적인 양식, 움직이는 이미지, 상호 작용적인 지도 등을 추가 할 수 있습니다.
브라우저 확장 프로그램, 모바일 앱, 데스크톱 앱도 만들 수 있으며 HTML, CSS, 웹 프레임 워크 등과 결합하여 복잡한 웹 응용 프로그램을 만들 수 있습니다.
