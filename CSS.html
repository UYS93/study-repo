1. CSS란 무엇인가?

    2. CSS(Cascading Style Sheets)는 웹 페이지의 레이아웃, 스타일, 글꼴 등의 디자인을 관리하는 언어입니다.
    CSS는 HTML 코드와 분리되어 사용되어 태그들에 스타일을 적용하는 것을 도와줍니다.
    이로 인해 HTML 코드는 웹 페이지의 구조와 내용을 정의하고 CSS는 그것들을 디자인 하는데 사용되어 웹 페이지를 정리적으로 꾸며줍니다.

    CSS는 선택자를 이용하여 특정 HTML 요소에 스타일을 적용할 수 있고, 박스 모델을 통해 레이아웃을 설정할 수 있으며,
    기타 위치, 색상, 글씨체, 배경 등 다양한 속성들을 설정 할 수 있어 웹 페이지를 디자인하는데 활용되고 있습니다.
