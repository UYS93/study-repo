1. HTML이란 무엇인가?

    2. HTML(Hypertext Markup Language)은 웹 페이지와 웹 응용 프로그램을 만드는 표준 마크업 언어입니다.
    웹 브라우저는 웹 서버 또는 로컬 스토리지로부터 HTML 문서를 수신하여 멀티미디어 웹 페이지로 렌더링합니다.
    HTML은 웹 페이지의 구조를 의미론적으로 설명하며 문서의 모양에 대한 단서를 포함합니다.

    HTML 요소는 HTML 페이지의 구성 요소입니다. HTML 구조를 사용하면, 이미지 및 기타 객체(예: 대화형 양식)는 렌더링된 페이지에 포함될 수 있습니다.
    제목, 문단, 목록, 링크, 인용문 및 기타 항목과 같은 텍스트에 대한 구조적 의미론을 표시하여 구조화된 문서를 만드는 수단을 제공합니다.
    HTML 요소는 태그로 설명되며, 각 괄호를 사용하여 작성됩니다.
